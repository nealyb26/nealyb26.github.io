TouchDesigner Flow Visualization

An Interactive Art Installation
Project Write-up Video Demo

Overview

This project is an interactive installation that creates real-time airflow visualizations around the silhouette of an object placed in the flow field. Using a depth camera system and projector mapping, the system will draw some beautiful 2d flow lines that respond dynamically to people's movements and positions.

I've always thought that aerodynamics testing yields some super striking imagery, so I wanted this to be a mix of that and an interactive art display. As of right now, the project is somewhat on hold, but I am looking to pick it back up over the summer

Technical Implementation

The system uses a depth camera for the real-time boundary estimation of objects. Contour detection algorithms identify the boundaries of people in the space, which are then used as obstacles in a simplified fluid dynamics simulation.

TouchDesigner serves as the primary platform for visual generation and projection mapping. The software receives silhouette data from the Python vision processing pipeline and generates smooth, flowing particle systems that simulate air currents deflecting around detected obstacles.

Projector calibration and mapping ensure that the visual effects align precisely with physical space, creating the illusion that airflow lines are drawn directly on the floor and respond naturally to people's movements.

Key Features

  • Real-time silhouette detection and tracking
  • Fluid dynamics simulation with obstacle avoidance
  • Smooth particle system visualization
  • Precise projector mapping and calibration
  • Multi-person tracking and interaction
  • Responsive visual feedback with minimal latency
  • Scalable installation design for different venues

Creative Process

The project began with explorations of how to make invisible forces visible in physical space. Wind and airflow are fundamental to our environment but remain largely unseen. By creating a digital representation that responds to human presence, the installation makes these invisible dynamics tangible and interactive.

The aesthetic draws inspiration from scientific flow visualization techniques used in wind tunnels and computational fluid dynamics research, translating these analytical tools into an engaging artistic medium that anyone can experience and understand intuitively.
